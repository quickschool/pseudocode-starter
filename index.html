Pseudocode Specification

Program Component Syntax Description Example
Declaration 
`name`, ..., `name-N` As `type`
Introduces unassigned variable identifier(s) of a specific data type to the current scope. 
X, Y, Z As Integer
Name As String
Sequence 
X <- Y
SayHello ()
Instructions are executed in order from top to bottom. 
Input (X)
Input (Y)
Sum <- X + Y
One Way Selection 
If `condition` Then
    `body:
End If
A condition is tested to determine if a branch or path is followed. 
If Age >= 18 Then
    Output ("Enterence Allowed")
End If
Two Way Selection 
If `condition` Then
    `true-body:
Else
    `false-body:
End If
A condition is tested and one branch or path is followed if the condition is true, and the other if the condition was false. 
If Age >= 18 Then
    Output ("Enterence Allowed")
Else
    Output ("Entry Denied")
End If
Multi Way Selection 
Case `expression` Of
    `condition-1`: `body-1`
    ...
    `condition-N`: `body-N`
End Case
A sequence of conditions are tested, the first condition (if any) to be evaluated as true will have the associated body of code executed. 
Case GradePct Of
    < 50   : GradeLetter <- F
    < 60   : GradeLetter <- P
    < 70   : GradeLetter <- C
    < 80   : GradeLetter <- D
    <= 100 : GradeLetter <- HD
End Case
For/Counted Loop 
For `index` <- `start` To `end`
    `body`
End For
A sequence of instructions is executed a fixed number of times with reference to a loop index variable. 
For GameNumber <- 1 To Length (Games)
    If Games[GameNumber].Win Then
        Output ("We won game #", GameNumber)
    Else
        Output ("We lost game #", GameNumber)
    End If
End For
While/Pre-Test Loop 
While `condition`
    `body`
End While
A condition is tested and if true a sequence of instructions are executed, this process is repeated a variable number of times. 
Answer <- 43
Guess  <- -1
While Guess != Answer
    Input (Guess)
End While
Repea Until/Post-Test Loop 
Repeat
    `body`
Until  `condition`
A sequence of instructions are executed a variable number of times until a condition is met. 
Answer <- 43
Repeat
    Input (Guess)
Until Guess == Answer
Function 
Function `function-name`(`parameter-1`, ..., `parameter-N`)
    `body`
    `function-name` <- `result-expression`
End Function
A function is a parameterized sequence of instructions which return a single result. 
Function LetterForGradePct(GradePct)
    Case GradePct Of
        < 50   : GradeLetter <- F
        < 60   : GradeLetter <- P
        < 70   : GradeLetter <- C
        < 80   : GradeLetter <- D
        <= 100 : GradeLetter <- HD
    End Case
    LetterForGradePct <- GradeLetter
End Function
Module 
Module `module-name`(`paramter-1`, ..., `parameter-N`)
    `body`
End Module

Call `module-name` (`argument-1`, ..., `argument-N`)
A module is a parameterized sequence of instructions which returns either zero or many results via reference parameters. A reference parameter allows a function to alter the argument variable itself. 
Module Initials(FirstName, LastName, FirstNameInitial, LastNameInitial)
    FirstNameInitial <- FirstName[0]
    LastNameInitial <- LastName[0]
End Module

Call Initials ("Luke", "Hollingworth", MyFirstNameInitial, MyLastNameInitial)
Output ("Initials are ", MyFirstNameInitial, ".", MyLastNameInitial)
Program Entrypoint 
Module Main
    `body`
End Module
The entrypoint of the program. 
Module Main
    Output ("Hello world")
End Module
Array 
`name` <- []
`name`[`index`] <- `expression`
Length (`name`)
An array is an ordered collections of values accessible by a numerical index starting at 0. The number of items of a list can be retrieved using the Length function. 
Primes <- [2, 3]
Primes[2] <- 5  // [2, 3, 5]
SumOfPrimes <- 0
For Index <- 0 To Length (Primes)
    SumOfPrimes <- SumOfPrimes + Primes[Index]
End For
Output ("Sum of primes = ", SumOfPrimes)
Record 
Record structure
    `record-name`
        `field-name-1`
        ...
        `field-name-N`
A record structure defines a new type of value which contains some number of fields accessible by name. 
Record structure

Record structure
        HttpResponse

            StatusCode

Example Programs

Distance Calculator

A function which calculates the distance between two points

Record structure
    Point
        X
        Y

Function Distance(Point1, Point2)
    DeltaX <- Abs (Point1.X - Point2.X)
    DeltaY <- Abs (Point1.Y - Point2.Y)
    Distance <- Sqrt (DeltaX^2 + DeltaY^2)
End Function

BMI Calculator

Calcultes the BMI of a list of people.

Function Bmi(MassKg, HeightM)
    Bmi <- MassKg / (HeightM^2)
End Function

Record structure
    Person
        Name
        MassKg
        HeightM

Module EnterPerson(Collection, Index, ShouldExit)
    Input (ShouldExit)
    If ShouldExit == False Then
        Input(NewPerson.Name)
        Input(NewPerson.MassKg)
        Input(NewPerson.HeightM)
        Collection[Index] <- NewPerson
    End If
End Module

Module Main
    People As []
    NumberOfPeople <- 0
    StopEnteringPeople <- False
    Repeat
        Call EnterPerson(People, NumberOfPeople, StopEnteringPeople)
    Until StopEnteringPeople

    For Index <- 0 To NumberOfPeople
        Person <- People[Index]
        PersonsBmi <- Bmi(Person.MassKg, Person.HeightM)
        Output(Person.Name, "has a BMI of", PersonsBmi)
    End For
End Module
